# mica-tinylog 组件

## 说明
`tinylog` 组件。支持 Spring boot 配置，支持启动完毕后关闭 Console，

## 使用
### maven
```xml
<dependency>
  <groupId>net.dreamlu</groupId>
  <artifactId>mica-tinylog</artifactId>
  <version>${version}</version>
</dependency>
```

### gradle
```groovy
compile("net.dreamlu:mica-tinylog:${version}")
```
